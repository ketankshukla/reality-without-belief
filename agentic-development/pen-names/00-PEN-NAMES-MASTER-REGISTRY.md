## Master Rules (All Pen Names)

1. **Author field**: Always use the pen name. Never "Ketan Shukla."
2. **Publisher**: Always "Metronagon Media"
3. **Standalone only**: No cliffhangers, no "Book 1 of...", no unresolved arcs
4. **Genre purity**: Each pen name stays in their lane. No genre mixing.
5. **Copyright**: © [Year] [Pen Name]. Published by Metronagon Media.
6. **KDP AI Disclosure**: Check the box on every submission
7. **No cross-references**: Books never reference other books

## Why These Names Work

### Phonetic Design Principles

Each name was selected for:
- **Rhythm**: 1-2 syllable first name + 2-3 syllable surname
- **Mouth feel**: Pleasing consonant combinations, easy to say aloud
- **Memorability**: Distinctive without being gimmicky
- **Genre fit**: Name "sounds like" the genre they write

### Name-by-Name Breakdown

| Name | Why It Works |
|------|--------------|
| **Kael Varen** | Hard K opening, short punchy syllables — sounds like a warrior wrote it |
| **Sera Nightingale** | Soft, flowing, musical — perfect for romantic fantasy |
| **Cole Harker** | Strong, no-nonsense, military bearing in the sound itself |
