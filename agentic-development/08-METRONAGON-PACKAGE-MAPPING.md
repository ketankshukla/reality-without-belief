## Workflow by Package

### Front Cover Packages

```
1. Receive brief (book concept, preferences)
2. Generate cover prompts (2-3 concepts)
3. Client selects preferred concept
4. Generate images using prompts
5. Deliver polished final
```

### Series Branding Packages

```
1. Receive brief (series concept, all book info)
2. Create series brand guide
3. Generate all cover prompts (consistent style)
4. Generate logo prompts
5. Generate box set prompt
6. Client reviews and selects
7. Generate images
8. Deliver complete package
```

### Full Pipeline Packages

```
1. Receive brief (series concept, all book info)
2. Create series brand guide
3. Generate all cover prompts
4. Generate logo prompts
5. Generate box set prompt
6. Generate all book descriptions
7. Research KDP categories/keywords
8. Client reviews and selects
9. Generate images
10. Deliver complete package
```

### Full Book Generation

```
1. Receive complete brief
2. Create story architecture
3. Generate all content (front matter → chapters → back matter)
4. Generate summaries and descriptions
5. Generate all image prompts
6. Generate Amazon assets
7. Generate audiobook prep
8. Generate marketing materials
9. Client review
10. Revisions if needed
11. Deliver complete project
```
